🎯 Mục tiêu dự án

Phát triển Web App truyền thông đa phương tiện cho doanh nghiệp bán lẻ, quản trị nội dung số, xuất bản đa kênh (Web/Facebook/Zalo/TikTok/YouTube), hỗ trợ livestream/clip ngắn, tối ưu SEO/Analytics; áp dụng SDLC (Waterfall/Agile-Scrum).

🔑 Tính năng chính

  • Quản trị CMS: bài viết, ảnh, video, lịch xuất bản, phân quyền người dùng
  • Livestream/WebRTC, hỗ trợ clip ngắn
  • Nhúng YouTube/TikTok
  • Social login (đăng nhập mạng xã hội)
  • Bình luận, duyệt nội dung
  • Trang chiến dịch marketing
  • Tối ưu Core Web Vitals, SEO, Analytics

🛠️ Công nghệ sử dụng

  • Frontend: Vue.js
  • Backend: Node.js
  • Database: MySQL

🚀 Hướng dẫn chạy demo

  1. Clone repo về máy
  2. Cài đặt dependencies cho frontend và backend
  3. Cấu hình database MySQL
  4. Chạy backend (npm start hoặc node index.js)
  5. Chạy frontend (npm run serve)
  6. Truy cập web app tại http://localhost:8080
